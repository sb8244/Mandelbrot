Mandelbrot by sb8244

Generator and explorer for the Mandelbrot set

Java desktop application which will plot any view of the Mandelbrot set, complete with the ability to change color scheme, view, image size, iteration count and the ability to save the current view as a picture.

Dependencies

Jave Runtime Environment
http://www.oracle.com/technetwork/java/javase/downloads/index.html

Install

Download the Mandelbrot.jar file and run it via command line with "java -jar Mandelbrot.jar" or just double click it (assuming you have Java set up correctly).
